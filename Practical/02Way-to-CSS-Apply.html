Way to CSS Apply

3 way to CSS Apply

  1. Inline CSS
  2. Internal CSS
  3. External CSS


Inline CSS => is css ko ham html document me direct body section me tag par use karte hai

Internal CSS => is css ko ham html document ke head section me use karte hai.

External CSS => is css ko ham html document me use nahi karte hai is ko ham alag se ek css file banate hai or us me use karte hai.
